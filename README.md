# 🥟 Sistema de Gestión de Empanadas

Este proyecto es un sistema desarrollado en **Python** que permite gestionar un menú de empanadas desde la terminal.  
El programa permite **listar, agregar, editar y eliminar empanadas**, guardando la información en un archivo JSON.

---

## 📌 Características

- Listar empanadas registradas
- Agregar nuevas empanadas
- Editar empanadas existentes
- Eliminar empanadas
- Guardar los datos automáticamente en un archivo JSON
- Menú interactivo en consola

---

## 🛠️ Tecnologías utilizadas

- **Python 3**
- **JSON** para almacenamiento de datos

---

## 📂 Estructura del proyecto

```
empanadas/
│
├── main.py
├── empanadas.json
└── README.md
```

- **main.py** → Archivo principal del programa  
- **empanadas.json** → Archivo donde se guardan las empanadas registradas  
- **README.md** → Documentación del proyecto  

---

## ⚙️ Funcionamiento

El programa funciona con un **menú interactivo en la terminal** donde el usuario puede elegir una opción:

```
1. Listar empanadas
2. Agregar empanada
3. Editar empanada
4. Eliminar empanada
5. Salir
```

Cada empanada contiene la siguiente información:

- **Nombre**
- **Precio**
- **Ingredientes**
- **Disponibilidad**

Todos los datos se guardan en el archivo **empanadas.json**.

---

## ▶️ Cómo ejecutar el proyecto

1. Tener **Python 3 instalado**.
2. Descargar o clonar el proyecto.
3. Ejecutar el programa en la terminal:

```bash
python main.py
```
